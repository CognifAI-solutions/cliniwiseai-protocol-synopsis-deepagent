# Generating Bioanalytical Procedures section.

The content for this section is standar and provided by the Bioanalytical site of each CRO. If the user has not provided this data, generate the section title and only add the following statement:

```
To be provided by the Bioanalytical site
```

If the user has provided the necessary information for this section, then expand on that content to generate the following subsections.

- Method Validation
- Assay of Samples
- Incurred Sample Reanalysis

## Generating Method Validation
Generate this subsection from the provided Bioanalysis

## Generating Assay of Samples
Add the following static content

```
- All available samples of the patients will be analyzed by the bioanalytical laboratory.
- Patients who have been withdrawn from the study due to adverse events will be analyzed for safety assessments only.

Patient samples will be analyzed using a calibration curve with quality control samples distributed throughout each batch. The details of the preparation of the calibration curve and quality control samples and the analytical batch acceptance criteria will be discussed in the respective bioanalytical study plan. The analytical staff will not have access to the randomization schedule.

If any concomitant medication is consumed by an evaluable patient during the clinical phase of the study, selectivity of the method will be checked in the presence of the concomitant medications.

The criteria for reanalysis and reintegration of the sample data will be applied as per in-house SOPs for **“Repeat Analysis”** and **“Chromatogram Acceptance Criteria”** respectively, and according to the guidelines set by the regulatory agencies.
```

## Generating Incurred Sample Reanalysis

Add the following static content for this section.

```
Incurred sample reanalysis evaluation and its compliance will be defined in bioanalytical plan and will be performed as defined in in-house standard operating procedure and in compliance to regulatory requirements.
```
