# Generating Study Schedule and Procedures.

This section contains the following sub-sections.
- Study Calendar
- Screening

# Generating Study Calendar Subsection
This section should have two Schedule of Assessments(SoA) tables in matrix format. Generate only empty matrices.

## 1. SoA table for visits from 1 to 3
- Use a matrix/table format.
- Rows should contain study evaluations/procedures.
- Columns should contain study visits or study periods.
- Include multi-level merged headers for study phases. The study phases are below:
    - Screening Part 01
    - Stabilization Period (at least 15 days prior to randomization)
    - Screening Part 02(Within 07 days prior to randomization)
- Include a visit number row beneath the headers.

These are the following assessments that have to added to the above table
- Informed consent 
- Inclusion and exclusion criteria
- Eligibility verification
- Demographics
- Patient characteristics
- Medical/disease history
- Medication history
- BRCA Test@#
- Physical examination
- Vital signs 
- ECOG status
- 12-lead ECG
- Chest X-Ray (postero-anterior view)
- If no thoracic imaging is performed within 06 month prior to screening
- Hematology:
        (total leukocyte count (WBC count), total erythrocyte count (RBC count), hemoglobin, HCT (PCV), lymphocytes, monocytes, granulocytes (neutrophils, basophils, eosinophils), platelet count,-absolute neutrophil count)
- Biochemistry:
        Random blood glucose, blood urea nitrogen (BUN), serum creatinine, total bilirubin, SGPT (ALT), SGOT (AST), ALP, sodium, calcium, potassium, chloride, magnesium, uric acid, phosphorous, lipid-profile (total cholesterol, LDL-cholesterol, triglycerides, HDL-cholesterol), D-dimer*
- Serology Test
- (HIV, Hepatitis B, HCV)
- Serum pregnancy test
- Urine pregnancy test
- Urine analysis
- Urine screen for drugs of abuse
- Urine alcohol test?
- Olaparib tablet (2*150 mg) 300 mg administration
- Patient diary dispensing
- Patient diary review
- Monitoring and capturing of AE/SAE
- Concomitant medications review and documentation

## 2. SoA table for visits from Day 4 to Day 14

- Use a matrix layout.
- Rows should contain evaluations/procedures.
- Columns should represent study periods, study days, and visits.
- Use hierarchical multi-row column headers with merged cells.
- Include grouped periods such as Treatment Period → Period 01 → Study Days.
- Include a Visit row beneath Study Day.

These are the assessments that have to be added to this table

- Eligibility verification
- ECOG status
- Physical examination
- Vital signs 
- Hematology:
    (total leukocyte count (WBC count), total erythrocyte count (RBC count), hemoglobin, HCT (PCV), lymphocytes, monocytes, granulocytes (neutrophils, basophils, eosinophils), platelet count, absolute -neutrophil count)
- Biochemistry:
    Random blood glucose, blood urea nitrogen (BUN), serum creatinine, total bilirubin, SGPT (ALT), SGOT (AST), ALP, sodium, calcium, potassium, chloride, magnesium, uric acid, phosphorous, lipid profile -(total cholesterol, LDL-cholesterol, triglycerides, HDL-cholesterol), D-dimer*
- Urine analysis
- Serum pregnancy test
- Urine pregnancy test
- Urine screen for drugs of abuse
- Urine alcohol test
- 12-lead ECG
- Randomization
- Investigational product (IP) administration4
- Hospitalization
- IP dispensing
- IP Accountability
- Patient diary dispensing
- Patient diary review
- PK sample collection9
- Monitoring and capturing of AE/SAE
- Concomitant medications review and documentation
