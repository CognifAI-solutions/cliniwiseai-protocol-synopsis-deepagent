# Generating Study Objectives and Rationale.

## Study Objectives.

The Study Objectives is already created in the protocol synopsis file at the below path in the file system. If the protocol synopsis file is missing, generate empty section for the Study Objective.
- **Protocol Synopsis**: /synopsis/protocol_synopsis.md

## Rationale
Derive the Rationale from the Study Objectives.
