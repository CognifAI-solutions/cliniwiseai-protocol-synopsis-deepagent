# Generating Handling of Investigational Product

Contains the following subsections

- Description of Investigational Product
- Investigational product receipt, storage, dispensing and accountability
- Compassionate medication to participating patients

## Generating Description of Investigational Product
Can be obtained from the protocol synopsis file that is available in the below file path. If the protocol synopsis file is missing, generate empty section for the Study Objective.
**Protocol Synopsis**: /synopsis/protocol_synopsis.md
  - Test Drug(T) in table format
  - Reference Drug(R) in table format

## Generating Investigational product receipt, storage, dispensing and accountability
A general write up, covering the Product receipt, storage, dispensing and accountability.

## Generating Compassionate medication to participating patients
Add the below static content for this subsection
```
As per the Investigator’s discretion, test product will be provided as a compassionate medication for up to 6 months or disease progression whichever occurs early.
```
